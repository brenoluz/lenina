# Lenina API Reference

Complete API reference for Lenina - Anvil RESTful Management.

## Base URL

- **Local:** `http://localhost:8000`
- **Docker:** `http://localhost:8000` (configurable)

## Authentication

No authentication required. Lenina is designed for local development only.

## Endpoints

### Health Check

```http
GET /health
```

**Description:** Health check endpoint for monitoring and container health checks.

**Response:** `200 OK`
```json
{
  "status": "healthy",
  "timestamp": "2026-03-04T10:30:00.000Z"
}
```

---

### Start Anvil

```http
POST /anvil/start
Content-Type: application/json
```

**Description:** Start a new Anvil instance with optional configuration.

**Request Body:** (all fields optional)
```json
{
  "port": 8545,
  "chainId": 31337,
  "blockTime": 0,
  "gasLimit": 30000000,
  "mnemonic": "custom mnemonic phrase here"
}
```

**Request Fields:**

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `port` | integer | `8545` | Anvil RPC port |
| `chainId` | integer | `31337` | Chain ID for the blockchain |
| `blockTime` | integer | `0` | Block time in seconds (0 = auto-mine) |
| `gasLimit` | integer | `30000000` | Gas limit per block |
| `mnemonic` | string | (auto) | HD wallet mnemonic phrase |

**Response:** `200 OK`
```json
{
  "pid": 12345,
  "port": 8545,
  "chainId": 31337,
  "status": "running"
}
```

**Error Responses:**

`400 Bad Request` - Anvil already running
```json
{
  "detail": "Anvil is already running"
}
```

`500 Internal Server Error` - Failed to start
```json
{
  "detail": "Anvil not found. Ensure Foundry is installed."
}
```

### Stop Anvil

```http
POST /anvil/stop
```

**Description:** Stop the currently running Anvil instance.

**Response:** `200 OK`
```json
{
  "status": "stopped",
  "message": "Anvil instance (PID 12345) has been stopped"
}
```

**Error Responses:**

`400 Bad Request` - No instance running
```json
{
  "detail": "No Anvil instance is running"
}
```

---

### Restart Anvil

```http
POST /anvil/restart
Content-Type: application/json
```

**Description:** Restart Anvil instance. Stops the current instance (if running) and starts a new one.

**Request Body:** (all fields optional, same as `/anvil/start`)
```json
{
  "port": 8545,
  "chainId": 31337,
  "blockTime": 0,
  "gasLimit": 30000000,
  "mnemonic": "custom mnemonic"
}
```

**Note:** If no configuration is provided and an instance is running, the previous configuration is preserved.

**Response:** `200 OK`
```json
{
  "pid": 12346,
  "port": 8545,
  "chainId": 31337,
  "status": "running",
  "message": "Anvil instance restarted successfully"
}
```

### Get Status

```http
GET /anvil/status
```

**Description:** Get the current running status of Anvil.

**Response (running):** `200 OK`
```json
{
  "running": true,
  "pid": 12345,
  "uptime": 123.45,
  "port": 8545
}
```

**Response (stopped):** `200 OK`
```json
{
  "running": false,
  "pid": null,
  "uptime": null,
  "port": null
}
```

**Response Fields:**

| Field | Type | Description |
|-------|------|-------------|
| `running` | boolean | Whether Anvil is currently running |
| `pid` | integer/null | Process ID of Anvil instance |
| `uptime` | float/null | Uptime in seconds |
| `port` | integer/null | Anvil RPC port |

    }
    // ... 8 more accounts

  "mnemonic": "test test test test test test test test test test test junk"

| Field | Type | Description |

### Check Contract

```http
GET /anvil/contract/{address}
```

**Description:** Check if a contract exists at the specified address.

**Path Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `address` | string | Ethereum contract address (0x-prefixed) |

**Response (contract exists):** `200 OK`
```json
{
  "address": "0x5FbDB2315678afecb367f032d93F642f64180aa3",
  "bytecodeHash": "0xabc123def456...",
  "deploymentBlock": null,
  "bytecode": "0x608060405234801561001057600080fd5b..."
}
```

**Response Fields:**

| Field | Type | Description |
|-------|------|-------------|
| `address` | string | Contract address |
| `bytecodeHash` | string | SHA256 hash of bytecode |
| `deploymentBlock` | integer/null | Block number (null if not tracked) |
| `bytecode` | string | Contract bytecode |

**Error Responses:**

`400 Bad Request` - Invalid address or Anvil not running
```json
{
  "detail": "Invalid Ethereum address format"
}
```

`404 Not Found` - No contract at address
```json
{
  "detail": "No contract deployed at address 0x..."
}
```

### Proxy RPC

```http
POST /anvil/rpc
Content-Type: application/json
```

**Description:** Proxy JSON-RPC requests to Anvil. Supports all standard Ethereum JSON-RPC methods.

**Request Body:** JSON-RPC 2.0 format
```json
{
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": [],
  "id": 1
}
```

**Example Methods:**

**Get block number:**
```json
{
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": [],
  "id": 1
}
```

**Get balance:**
```json
{
  "jsonrpc": "2.0",
  "method": "eth_getBalance",
  "params": ["0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266", "latest"],
  "id": 1
}
```

**Send transaction:**
```json
{
  "jsonrpc": "2.0",
  "method": "eth_sendTransaction",
  "params": [{
    "from": "0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266",
    "to": "0x70997970C51812dc3A010C7d01b50e0d17dc79C8",
    "value": "0x16345785d8a0000"
  }],
  "id": 1
}
```

**Response:** `200 OK`
```json
{
  "jsonrpc": "2.0",
  "result": "0x1",
  "error": null,
  "id": 1
}
```

**Error Responses:**

`400 Bad Request` - Anvil not running or RPC error
```json
{
  "detail": "No Anvil instance is running"
}
```

**Supported Methods:**
- `eth_blockNumber` - Get current block number
- `eth_getBalance` - Get account balance
- `eth_getCode` - Get contract bytecode
- `eth_call` - Execute contract call (read-only)
- `eth_sendTransaction` - Send transaction
- `eth_sendRawTransaction` - Send signed transaction
- `eth_getTransactionReceipt` - Get transaction receipt
- `eth_getLogs` - Get event logs
- `eth_getStorageAt` - Get contract storage
- `eth_estimateGas` - Estimate gas for transaction
- `eth_gasPrice` - Get current gas price
- `eth_accounts` - Get available accounts
- And all other standard Ethereum JSON-RPC methods

## Error Handling

All errors follow the FastAPI standard format:

```json
{
  "detail": "Error message describing what went wrong"
}
```

### HTTP Status Codes

| Code | Meaning | When |
|------|---------|------|
| `200` | OK | Request succeeded |
| `400` | Bad Request | Invalid request (Anvil not running, invalid params) |
| `404` | Not Found | Contract not found at address |
| `500` | Internal Server Error | Server error (Anvil failed to start) |
